SimJQ

——4K不到的嵌入式JS框架

SimJQ全称Simple JQuery，支持类似JQuery语法的常用DOM操作和网络请求功能。它不到4K大小，适合应用于存储空间极其有限的场景。

下载 simJQ-1.3.js

或直接引用： <script type="text/javascript" src="//leytton.gitee.io/simjq/simJQ-1.3.min.js"></script>

eg98:ajax get网络请求

var url='http://www.llqqww.com/open/json/get.php';
$.ajax({
   	method: 'GET',
   	dataType:'json',
   	url:url,
    data:{
    	name:'Leytton'
    },
    success: function (data,xhr) {
       alert(JSON.stringify(data));
    },
    error:function(data,xhr){
        alert(data);
    }
});
